Descripción del sistema

La página es un pequeño sistema basado en HTML, CSS y JAVA, creado como un proyecto en Maven; su objetivo es dar a conocer el trabajo de nuestra clienta Patricia Páez Huerta, a su vez el dar a la venta los distintos productos disponibles en los catálogos, dichos productos proporcionados para la misma página, siendo decoraciones para fiestas infantiles, de XV y día de muertos; se ha implementado un sistema de ventas donde en dichos catálogos pueden elegir productos a su gusto, agregarlos a un carrito, deberán llenar un formulario con información esencial para poder hacer la compra, por ejemplo, su nombre, dirección para envió, un correo y numero de celular para contacto y con ello se dará información de coste según los productos y su envío, al finalizar la información se guardara en una base de datos en MongoDB donde lo podrá consultar la dueña.

Este sistema es creado para la materia de Sistemas Web en conjunto con Interacción Humano Computadora, con un objetivo educativo para la práctica del desarrollo de sistemas web.

Manual de usuario

1. Navegar por Inicio/Home

Debe abrir la página web desde el siguiente enlace: https://colleage.herokuapp.com/

Lo primero que vera es una página de inicio que lo recibe con un mensaje, en esta página no se usa la barra de navegación, solo haga clic y avanzara por el contenido de la página.

2. Ir a Contacto/Contact

Para ir a la página Contact debe dirigirse a la parte superior derecha al botón enlace llamado Contact.

Dentro de Contact solo existe información de contacto de la propietaria del negocio.

3. Ir a Diseños/Designs

Para ir a la página de Designs existen dos formas

3.1. Por medio de la barra de navegación

En la parte superior derecha en el botón enlace llamado Designs.

3.2. Por medio de Inicio/Home

Puedes llegar a la parte final de la página de Home y dar en el texto “Check it our designs”.

4. Entrar a un catalogo

Dentro de la página Designs existirán 3 apartados:

* Fiesta infantil

* XV Años

* Día de muertos

Para entrar a cualquier apartado debe hacer clic en el texto del apartado que desea visitar.

Dentro de cada apartado se encontrará con un catálogo para poder elegir decoraciones que desee comprar.

5. Comprar uno o varios productos

Si desea comprar algo debe entrar al catalogo de su interés.

Si desea comprar algo debe ver el catálogo.

Si encontró algo de su agrado debe elegir la cantidad de productos que desea comprar de dicho producto.

Dar clic en Add y se agregara a su carrito.

Para ver su carrito solo debe hacer clic en el icono del carrito de comprar encontrado en la parte superior derecha.

Si su carrito se encuentra vacío, este le avisará en caso de que intente acceder además de que el icono no tendrá su modificación que avisa que existe algo dentro del carrito.

Una vez agregados productos al carrito, lo abrimos y nos aparecerá un resumen de los productos que agregamos, para continuar daremos “clic” en siguiente.

Nos llevará a esta pantalla en la cual deberemos ingresar el nombre completo y correo electrónico de quien recibirá las decoraciones. Una vez ingresados daremos “clic” en siguiente.

Esta vez deberemos ingresar dirección completa, código postal y teléfono móvil de la persona que antes ingresamos. Una vez ingresados daremos “clic” en siguiente.

Ahora deberemos introducir número de tarjeta, fecha de expiración y CVV de alguna tarjeta de débito o crédito.

(Para más detalles de la ubicación de estos elementos revisar Anexo)

Una vez ingresados todos los datos correctamente daremos “clic” en comprar.

Nos aparecerá un mensaje confirmando que la compra se realizó con éxito.

¡Gracias por su preferencia!

Manual de instalacón

Para instalar el código necesita los siguientes prerrequisitos:

* Maven >= 3.6.0

* Java JDK >= 1.8

* Cuenta de MongoDB si desea usar una base de datos propia siga las siguientes instrucciones:

 Entre a su cluster

 Entre en la opción de colecciones

 Cree 5 bases de datos/colecciones llamadas Dia De Muertos, Eventos, Fiesta Infantil, Orders y VX

 Dentro de día De Muertos, Fiesta Infantil y VX crear documentos con el siguiente tipo de datos:

- name(String) = es el nombre del producto

- type(String) = tipo de producto (Pared/Mesa/Suelo/Techo)

- size(String) = medidas del producto

- pieces(Int32) = número de piezas disponibles

- price(Int32) = precio del producto

- src(String) = URL de donde se encuentra la imagen del producto correspondiente

 Ejemplo:

 Dentro de Eventos crear documentos con el siguiente tipo de datos:

- text(String) = nombre del catálogo (Fiesta Infantil/ XV Años/Dia de Muertos)

- src(String) = URL de la ubicación sobre donde se encuentra la imagen que se usara de fondo

- ref(String) = lugar donde está el código del catálogo

- position(Int32) = número de lugar de la posición

 Ejemplo:

 Dentro de Orders solo se crearán por medio del código, pero los tipos de datos quedarían como:

- username(String) = se guardará el nombre del comprador

- email(String) = se guardará el correo del comprador

- address(String) = se guardará la dirección a donde se enviará

- cp(String) = se guardará el código postal

- phone(String) = se guardará el número de teléfono

- card(String) = se guardará el número de tarjeta del comprador

- expiration(String) = se guardará la fecha de expiración de la tarjeta

- cvv(String) = se guardará el código de la tarjeta

- ítems(Array) el cual tendrá dentro objetos con nombre numéricos donde dentro de ese objeto estarán los datos de los productos solicitados con datos como:

o name(String) = se guardará el nombre del producto

o cost(Int32) = se guardará costo del producto individual

o amount(Int32) = se guardará la cantidad solicitada del producto

o total(Int32) = se guardará el total de la compra de ese producto x la cantidad

 Ir a la carpeta del proyecto y dentro de la carpeta src -> main -> java -> com -> my -> app -> Connection en el archivo Mongo.java modificar el método connnect()

- poniendo la URL entre las comillas de la siguiente línea:

this.uri = new MongoClientURI(" ");

- Poniendo el nombre de tu proyecto/base de datos:

this.database = this.mongoClient.getDatabase(" ");

Una vez cubiertos estos prerrequisitos debemos tener Visual Studio Code, con ello se puede abrir el código

El procedimiento es sencillo, solo te ubicas en el directorio donde Guardarás el proyecto y lo clonas con el siguiente comando:

git clone https://github.com/Chekorita/Arreglos

Luego te ubicas en el directorio que se acaba de crear:

cd Arreglos

Y ejecutas el siguiente comando para compilar y correr:

mvn exec:java -D"exec.mainClass"="App"

En caso de falla poner el siguiente comando:

mvn compile && mvn exec:java -D"exec.mainClass"="App"
